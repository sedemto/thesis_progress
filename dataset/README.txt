**Dataset**

This folder is to contain datasets.
You can download and save any files here, and please accordingly change "soundDir" in main.m

The EBU SQAM database that was used in the experiments is available in https://github.com/ondrejmokry/InpaintingRevisited.
You can reproduce the results of PHAINs by downloading EBU_SQAM.mat from the URL and executing main.m with it.
